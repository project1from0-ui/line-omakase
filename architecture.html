たべコーチ - システム概要

LINE連携AI栄養指導プラットフォーム

アプリ概要

たべコーチは、パーソナルトレーナーがLINEを通じてクライアントの食事管理を行うためのプラットフォームです。 クライアントは普段使っているLINEで食事写真やテキストを送るだけで、AIが自動的に栄養素を解析し、パーソナライズされた指導を返します。 トレーナーはWebダッシュボードから全クライアントの栄養状況をリアルタイムで確認・介入できます。

主要機能

📸

AI食事画像認識

写真を送るだけでGemini AIが食品を自動認識し、カロリー・PFC（タンパク質/脂質/炭水化物）を推定

🤖

パーソナライズ指導

体重・目的・アレルギー・既往歴などの個人情報を考慮した、一人ひとりに合わせたAI栄養指導

📊

AI日次レポート

1日の食事を総括したAI総評を指定時刻に自動送信。テンプレートではなくAIが毎日生成

📖

LIFF食事日記

LINE内ブラウザで過去の食事記録・栄養達成率・AI総評を日別カード形式で閲覧

🔔

未報告リマインダー

12時間以上食事報告がないクライアントへ自動リマインド。メッセージはカスタマイズ可能

💬

トレーナー直接介入

ダッシュボードからクライアントへ直接メッセージ送信。チャット履歴も一元管理

🎯

栄養目標自動計算

身長・体重・活動レベル・目的から最適なカロリー・PFC目標を自動算出

📱

PWAダッシュボード

トレーナー向けWebアプリ。クライアント一覧、栄養推移グラフ、設定をスマホからも操作

利用者

トレーナー（管理者）
  • Webダッシュボードでクライアント管理
  • 栄養目標・個人情報の設定
  • AIシステムプロンプトのカスタマイズ
  • リマインド・レポート時刻の設定
  • チャット履歴の閲覧・直接メッセージ
クライアント（顧客）
  • LINEで食事写真・テキストを送信
  • AIからリアルタイムで栄養指導を受信
  • 日次レポートで1日の振り返り
  • LIFF食事日記で過去記録を閲覧
  • 追加アプリのインストール不要

システム全体構成図

主要データフロー

1. 食事報告 → AI指導（リアルタイム）
クライアント: 食事写真送信 → LINE Messaging API → lineWebhook → Gemini: 画像認識+栄養推定 → Firestore: 保存+日次累計更新 → LINE: AI指導を返信
2. 日次AIレポート（スケジュール実行）
cron: 毎時0分 → summaryTime一致のテナント抽出 → 当日の全食事取得 → Gemini: AI総評生成 → dailySummaries保存 → LINE: レポート+日記リンク送信
3. 食事日記閲覧（LIFF）
クライアント: レポート内リンクをタップ → LINEアプリ内ブラウザ (LIFF) → LIFFトークン取得 → getDailySummaries API → 日別カード表示
4. 未報告リマインド（スケジュール実行）
cron: 毎日21:00 JST → 12時間以上未報告ユーザー抽出 → LINE: リマインドメッセージ送信 → ダッシュボード通知作成

Firestoreデータ構造

パス内容
tenants/{tenantId}Bot設定 (LINE認証情報, システムプロンプト, summaryTime)
tenants/{tenantId}/users/{userId}クライアント情報 (プロフィール, personalInfo, nutritionalGoal, 日次栄養累計)
tenants/{tenantId}/users/{userId}/messages/{msgId}チャット履歴 (user/ai/trainer, テキスト/画像, 栄養データ)
tenants/{tenantId}/users/{userId}/dailySummaries/{date}日次レポート (PFC合計, 食事数, AI総評, 目標スナップショット)
tenants/{tenantId}/notifications/{id}トレーナー向け通知 (未報告アラート等)
